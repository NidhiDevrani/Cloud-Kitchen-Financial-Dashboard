## Overview

This project is an interactive financial analytics dashboard built for a cloud kitchen business. The dashboard helps analyze Profit & Loss (PNL) performance across different kitchen stores, cities, and months by providing key financial metrics, operational insights, and interactive visualizations.

The goal of this project is to transform raw financial data into meaningful business insights that can support decision-making and performance monitoring.

---

## Business Problem

Cloud kitchen businesses operate multiple kitchen locations across different cities. Tracking profitability and operational performance across these locations can be challenging when data is stored in spreadsheets.

This dashboard provides a centralized view of:

* Revenue performance
* Gross Margin analysis
* EBITDA tracking
* Store-level comparisons
* City-level performance
* Monthly trends
* Operational metrics such as order volume

By using dynamic filters, users can quickly explore financial performance across various business dimensions.

## Features

### Interactive Filters

Users can filter data by:

* Month
* Store
* City
* Revenue Cohort
* EBITDA Category
* EBITDA Cohort
* CM Cohort
* EBITDA Range
* Gross Margin Range
* Gross Margin Percentage Range
* NET Revenue Range

All visualizations and KPIs update dynamically based on selected filters.

---

### KPI Cards

The dashboard displays key business metrics:

* Total Revenue
* Total EBITDA
* Total Orders
* Store Count

These KPIs provide a quick overview of business performance.

---

### Interactive Data Table

Users can:

* View filtered records
* Sort data
* Explore store-level performance
* Analyze financial metrics


---

### Visual Analytics

#### Revenue by Store

Bar chart showing revenue contribution from each store.

#### Monthly EBITDA Trend

Line chart tracking EBITDA performance over time.

#### Revenue Contribution by City

Donut chart visualizing city-wise revenue share.

#### Revenue vs EBITDA Analysis

Scatter plot comparing revenue and EBITDA across stores (can filter by City).

Users can download filtered datasets for further analysis.
